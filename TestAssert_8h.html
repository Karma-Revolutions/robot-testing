Conditional failure report.

RTF_ASSERT_CHECK does not throw any exception. It reports a failure message to the TestResult indiecating that the check on the corresponding condition evaluates to false.

Parameters
messageMessage to be reported as the detail of TestMessage
conditionIf this condition evaluates to false then the test failed.
Note
RTF_CHECK throws error exception if it is not called within a TestCase class.
